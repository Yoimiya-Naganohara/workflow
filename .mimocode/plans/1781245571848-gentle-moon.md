# TUI Redesign — Implementation Plan

## Goal
将 TUI 从当前设计改为 opencode 风格 + ratatui 原生渲染。

## 核心设计决策

### 布局
```
┌──────────────────────────────────────────────────────────────────────────────┐
│                                                                              │
│  ┃ <user message>                                                            │
│                                                                              │
│  │ Tool: params                                                              │
│  │ result...                                                                 │
│                                                                              │
│  ┃ <agent message>                                                           │
│  ┃                                                                           │
│  ┃  ┌─ lang ─────────────────────────┐                                       │
│  ┃  │ code...                        │                                       │
│  ┃  └────────────────────────────────┘                                       │
│                                                                              │
│  │ Tool: params                                                              │
│  │ + diff line                                                               │
│                                                                              │
│  ⠋ Generating...                                                             │
│                                                                              │
├──────────────────────────────────────────────────────────────────────────────┤
│ > input_                                                                     │
├──────────────────────────────────────────────────────────────────────────────┤
│ claude-sonnet-4 ↑5.5K ↓1.2K pool:128 72%    ✖2 ⚠3    Claude Sonnet 4       │
└──────────────────────────────────────────────────────────────────────────────┘
```

### 消息渲染规则

| 类型 | 左栏颜色 | 样式 |
|------|----------|------|
| User | 绿色 (thick) | `┃` + 缩进文本 |
| Agent | 蓝色 (thick) | `┃` + 缩进文本 |
| Tool | 灰色 (thick) | `│` + `ToolName: params` + 结果 |
| Thinking | 无 | `⠋ Generating...` (spinner) |

### Status Bar（opencode 风格）
```
[help widget] [tokens widget] [spacer] [diagnostics] [model widget]
```
- help: `ctrl+? help` — `text2` bg, `bg3` fg
- tokens: `Context: 5.5K, Cost: $0.12` — `text` bg, `bg2` fg
- diagnostics: `✖2 ⚠3` — `bg3` bg, red/yellow fg
- model: `Claude Sonnet 4` — `mauve` bg, `bg3` fg

### 配色（Catppuccin Mocha）
```
bg:     #212121    text:   #cdd6f4    blue:   #89b4fa
bg2:    #2c2c2c    text2:  #a6adc8    mauve:  #cba6f7
bg3:    #181818    text3:  #7f849c    green:  #a6e3a1
border: #4b4c5c    overlay0:#6c7086   red:    #f38ba8
                                            yellow: #f9e2af
```

## 实现步骤

### Step 1: style.rs — 新色板
```rust
// 新增 Catppuccin Mocha 颜色常量
pub const BG: Color = Color::Rgb(33, 33, 33);
pub const BG2: Color = Color::Rgb(44, 44, 44);
pub const BG3: Color = Color::Rgb(24, 24, 24);
pub const TEXT: Color = Color::Rgb(205, 214, 244);
pub const TEXT2: Color = Color::Rgb(166, 173, 200);
pub const TEXT3: Color = Color::Rgb(127, 132, 156);
pub const BLUE: Color = Color::Rgb(137, 180, 250);
pub const MAUVE: Color = Color::Rgb(203, 166, 247);
pub const GREEN: Color = Color::Rgb(166, 227, 161);
pub const RED: Color = Color::Rgb(243, 139, 168);
pub const YELLOW: Color = Color::Rgb(249, 226, 175);
pub const OVERLAY0: Color = Color::Rgb(108, 112, 134);
```

### Step 2: chat_lines.rs — 核心重写
- `build_chat_lines()` → 返回 `Vec<Line<'static>>`
- User: `Style::default().fg(GREEN)` thick bar + 缩进
- Agent: `Style::default().fg(BLUE)` thick bar + 缩进
- Tool: `Style::default().fg(OVERLAY0)` thick bar + `ToolName: params`
- Thinking: spinner animation + `Generating...`
- Code block: 2空格缩进，无边框
- Diff: 绿色 `+` 前缀

### Step 3: render.rs — 全宽 Chat
- 移除 Proposal 面板
- Chat 全宽
- Status bar 一行

### Step 4: status.rs — opencode 风格
- help | tokens | spacer | diagnostics | model
- 每个 widget 用 `Block::default().borders(Borders::ALL)` + 背景色

### Step 5: chat.rs — 无边框 Input
- 只有 `>` 提示符 + textarea
- 移除 border + label

### Step 6: state.rs — 移除 Proposal 状态

## 文件清单
