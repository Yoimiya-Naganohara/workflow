## Status: Exploration Complete

The user requested "查看当前的经验系统" (view the current experience system). The agent completed a thorough read-through of all 5 files in `src/experience/`. This was an inspection request — no code changes were made.

## Key Findings

The experience system consists of:

1. **`pool.rs`** — Mmap-backed persistent A-track (bedrock). 64-byte header, default 1024 entries, grows by 512. Weighted cosine similarity search.
2. **`dual_track.rs`** — Dual-track memory combining bedrock + volatile fluid. Auto-consolidation when fluid exceeds threshold. Credibility weighting (bedrock 1.0, fluid 0.6). Implements `ExperienceRetrieval` trait.
3. **`clustering.rs`** — Leader clustering with Welford online algorithm. Default threshold 0.85 cosine similarity. Filters by min_cluster_size before promoting to bedrock.
4. **`role_template_store.rs`** — JSON-persisted role templates with exact + semantic lookup. Separate from mmap pool.
5. **`mod.rs`** — Module re-exports.
