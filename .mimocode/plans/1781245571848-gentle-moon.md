# TUI Redesign v2

## Core Insight

当前 TUI 的根本问题不是视觉，而是**信息架构**。每个消息都带着 status icon + role label + timestamp 头部，tool call 是独立的 message block，code block 有自己的 border/header。这导致屏幕上 60% 是装饰，40% 是内容。

**目标：让内容占据 80%，装饰只占 20%。**

## 新设计哲学

### 1. 消息流 = 连续的文本流

不是 "每条消息一个 block"，而是像 Claude Code 那样，agent 的回复是一个连续的文本流，tool call 内嵌在流中：

```
You: Implement auth middleware

Agent: I'll create the auth middleware.
  
  read_file src/models/user.rs
  
  Here's the implementation:

  ┌─ rust ─────────────────────────┐
  │ use axum::{Extract, Request};  │
  │ ...                            │
  └────────────────────────────────┘

  write_file src/middleware/auth.rs
  diff: +21, -0

  Done. Key decisions:
  • RwLock for concurrency
  • Claims via request extensions
```

**关键变化：**
- 去掉每条消息的状态图标（✓/●/◌）— 只在 streaming 时显示
- 去掉每条消息的 role label — 用缩进和颜色区分
- 去掉每条消息的 timestamp — 只在 status bar 显示时间
- Tool call 不是独立消息，是 agent 回复的一部分
- User 消息用蓝色左栏，Agent 消息无装饰

### 2. Tool Call = 内联引用

```
  read_file src/models/user.rs
```
- 一行，无边框，无装饰
- 紫色高亮 tool 名，灰色参数
- 输出只在需要时显示（错误时自动展开）

### 3. Code Block = 无边框

```
  use axum::{Extract, Request, Next};

  pub async fn auth_middleware(
      mut req: Request,
      next: Next,
  ) -> Result<Response, StatusCode> {
      // ...
  }
```
- 无边框，无 header，无行号
- 用 2 空格缩进 + 背景色区分
- 语言标签只在第一个 code block 显示

### 4. Diff = 内联

```
  src/middleware/auth.rs (+21, -0)
  + use axum::{Extract, Request, Next};
  + pub async fn auth_middleware(
  +     mut req: Request,
  + ) -> Result<Response, StatusCode> {
```
- 无边框，直接嵌入
- 绿色 + 红色 - 前缀

### 5. Status Bar = 极简

```
  gpt-4  ↑2.1k ↓3.4k  pool:128  budget:72%    Ctrl+A providers
```
- 一行，无装饰
- 左：模型 + token + pool + budget
- 右：键盘提示

### 6. Input = 无边框

```
  > Add integration tests_
```
- 只有一个 `>` 提示符
- 无边框，无背景
- 多行时自动扩展

## 具体改动

### style.rs
- 移除 `panel()`, `panel_chat()`, `panel_proposal()`, `input_bar()`
- 新增：`msg_user_bar()`, `tool_name_style()`, `code_indent_style()`
- 颜色：保持 Catppuccin 但减少使用

### chat_lines.rs — 核心重写
- 消息渲染为连续文本流
- Tool call 内嵌到 agent 消息中
- Code block 无边框，2空格缩进
- Diff 无边框，内联

### render.rs
- 移除 Proposal 面板
- Chat 全宽
- Input 无边框

### chat.rs
- Input box 改为无边框
- 移除 popup 区域（命令补全改为内联）

### status.rs
- 重新设计为一行极简状态

### popup.rs
- 命令补全改为 input 下方内联，无边框

## 文件清单

| 文件 | 改动 |
|------|------|
| `src/tui/style.rs` | 精简 color palette，移除 border widgets |
| `src/tui/chat_lines.rs` | **核心重写**：连续文本流 + 内嵌 tool call |
| `src/tui/render.rs` | 移除 Proposal，全宽 Chat |
| `src/tui/chat.rs` | 无边框 input |
| `src/tui/status.rs` | 极简一行状态 |
| `src/tui/popup.rs` | 内联命令补全 |
| `src/tui/state.rs` | 移除 Proposal 相关状态 |
